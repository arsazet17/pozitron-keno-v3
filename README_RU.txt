ПОЗИТРОН — Анализатор КЕНО v3

В комплекте:
• автоматическая проверка свежих тиражей Lucky Numbers при открытии;
• кнопка ручного обновления;
• добавление только недостающих тиражей;
• поиск по победному столбу, дате, диапазону дат, номеру и комбинации;
• выгрузка результатов в CSV;
• собственный значок-мишень.

ВАЖНО:
Версия v3 содержит серверную Netlify Function. Обычная замена одного HTML-файла через Netlify Drop не активирует автоматическую загрузку.
Нужно развернуть весь проект через GitHub → Netlify «Import an existing project» либо через Netlify CLI.

После развёртывания:
1. Один раз загрузите полный CSV.
2. Нажмите «Сохранить базу».
3. Далее при открытии анализатор проверяет свежие тиражи и дописывает их.
